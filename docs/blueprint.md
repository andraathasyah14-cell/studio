# **App Name**: INBIO Assistant

## Core Features:

- Text Improvement Tool: Use AI to rewrite the given text to improve clarity, tone, and impact, acting as a tool to make it better.
- Content Summarization: Generate concise summaries of the content of each feature to highlight the most important points, improving readability.
- Dynamic Feature Cards: Display features in an interactive and visually appealing card format, mimicking the existing layout, with consistent use of icons and clear descriptions.
- Animated Transitions: Incorporate subtle transitions and animations to improve the user experience and guide the user through the feature list.
- Interactive elements: Clickable elements to encourage interaction

## Style Guidelines:

- Primary color: Use a vibrant rose (#C91F53) to capture attention and reflect energy. This aligns with the pre-existing button styling in the request.
- Background color: Employ a very light gray (#F7F7F8) to keep a clean and professional look, giving prominence to other elements.
- Accent color: Use a light shade of purple (#B39DDB) to give some differentiation, as this hue resides near rose in the color space.
- Font: 'PT Sans', a sans-serif font, offers a clean, modern appearance suitable for both headlines and body text, maintaining readability and a professional tone.
- Employ a set of simple and modern icons, consistent with the design present in the request, using them to add visual cues.
- Maintain a clean and structured layout, utilizing white space effectively to make content accessible and engaging.
- Subtle fade-in and slide-up effects to enhance the interactive experience when features are displayed.
